# Cursor ショートカットコマンド一覧

Cursorは、AI機能を統合した強力なコードエディタです。以下に、生産性を向上させるための主要なショートカットコマンドをまとめました。

## AI関連機能

### Cursor Tab（AI コード補完）
- **コード補完を受け入れる**: `Tab`
- **コード補完を拒否する**: `Esc`
- **部分的に受け入れる**: `Ctrl/⌘ + →`

### Cmd K（インライン編集）
- **Cmd K を開く**: `Ctrl/⌘ + K`
- **変更を適用する**: `Ctrl/⌘ + Enter`
- **変更をキャンセル/削除する**: `Ctrl/⌘ + Backspace`

### チャットインターフェース
- **チャットを開く**: `Ctrl/⌘ + L`
- **コードをチャットに追加**: `Ctrl/⌘ + L`
- **チャットを閉じる**: AI サイドバーを閉じる

### Composer（複数ファイル編集）
- **Composer を開く**: `Ctrl/⌘ + I`
- **フルスクリーン Composer を開く**: `Ctrl/⌘ + Shift + I`
- **ファイルを追加**: `Ctrl/⌘ + P`（Composer内）

## @ シンボル（参照機能）
- **ファイルを参照**: `@filename`
- **関数を参照**: `@functionName`
- **変数を参照**: `@variableName`
- **コードベースを検索**: `@codebase query`
- **Webを検索**: `@web query`

## 一般的なショートカット

### ファイル操作
- **新しいファイル**: `Ctrl/⌘ + N`
- **ファイルを保存**: `Ctrl/⌘ + S`
- **ファイルを開く**: `Ctrl/⌘ + O`

### 表示・ナビゲーション
- **コマンドパレットを開く**: `Ctrl/⌘ + Shift + P`
- **設定を開く**: `Ctrl/⌘ + ,`
- **サイドバーを切り替え**: `Ctrl/⌘ + B`
- **ターミナルを切り替え**: `Ctrl/⌘ + \`` `
- **ファイルエクスプローラー**: `Ctrl/⌘ + Shift + E`

### 編集機能
- **検索**: `Ctrl/⌘ + F`
- **置換**: `Ctrl/⌘ + H`
- **ファイル内で検索**: `Ctrl/⌘ + Shift + F`
- **行を複製**: `Ctrl/⌘ + Shift + D`
- **行を削除**: `Ctrl/⌘ + Shift + K`
- **行を上/下に移動**: `Alt + ↑/↓`

## 設定関連

### キーボードショートカットのカスタマイズ
1. `Ctrl/⌘ + Shift + P` でコマンドパレットを開く
2. "Open Keyboard Shortcuts (JSON)" を検索
3. 設定を編集

### 主要なコマンド名
- **Cmd K**: `aipopup.action.modal.generate`
- **チャットを開く**: `workbench.action.chat.open`
- **AI サイドバーを閉じる**: `workbench.action.closeSidebar`

## 注意点
- **Windows/Linux**: `Ctrl` キーを使用
- **macOS**: `⌘` (Command) キーを使用
- ショートカットは設定でカスタマイズ可能
- 一部の機能は、選択したテキストの有無によって動作が変わる

## 使用のヒント
1. **Ctrl/⌘ + K** は選択したコードがあるかどうかで動作が変わります
   - 選択なし: 新しいコードを生成
   - 選択あり: 選択したコードを編集
2. **@** シンボルを使って、コンテキストを効率的に参照できます
3. **Composer** は複数のファイルを同時に編集する際に便利です

これらのショートカットを活用することで、Cursorでの開発効率を大幅に向上させることができます。
